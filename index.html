Sobre

Desenvolvido no curso de Engenharia de Software da Universidade de Vassouras, Maricá - RJ, o PetHealth é uma plataforma web projetada para otimizar o gerenciamento da saúde de animais de estimação. O aplicativo permite que proprietários registrem e acompanhem consultas, tratamentos, vacinações e recebam lembretes automatizados, facilitando a comunicação com veterinários e o monitoramento do bem-estar dos pets.

Utilizando tecnologias modernas como Node.js, JavaScript, HTML, CSS e MongoDB, o PetHealth foi desenvolvido com metodologias ágeis, como Scrum, garantindo flexibilidade e adaptação às necessidades do usuário. Este projeto representa uma aplicação prática e inovadora, com o potencial de impactar positivamente a vida de animais e seus proprietários.
